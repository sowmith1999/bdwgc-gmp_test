# bdwgc-gmp_test

## Getting Started

## GMP
- get the tar from GMP [here](https://ftp.gnu.org/gnu/gmp/)
- extract it
- use "../../gmp-6.2.1/configure --enable-cxx --prefix="<PATH TO GMP>/gmp_trail/dependencies/"
- make
- make check
- make install
  
## BDWGC
- same as gmp, except get the file [here](https://www.hboehm.info/gc/#where)
